Software Design & Engineering

Restaurant Mobile App Enhancement - Demonstrating UI/UX design, component architecture, and mobile development best practices

About the Artifact

The artifact I selected is a mobile restaurant information app created in the CS 360 course, "Mobile Architecture and Programming." It was built using React Native and Firebase. The app allows users to view restaurant lists, see menu items, and log in. I designed the user interface and implemented the main screens.

I chose this artifact because it shows my ability to create a user-friendly mobile app and connect it with a backend system. Originally, the app did not have a reservation feature, so I planned to enhance it by adding a new screen for users to make reservations.

Before Enhancement

  • Basic restaurant listing view
  • Simple menu display
  • Basic user authentication
  • Static content only
  • Limited user interaction

After Enhancement

  • Comprehensive reservation system
  • Review submission functionality
  • Favorites management
  • Enhanced UI/UX design
  • Improved user experience

Enhancement Implementation

Reservation System

I implemented a comprehensive reservation system that allows users to:

  • Select restaurants from the main listing
  • Choose preferred date and time slots
  • Specify party size and special requests
  • Submit reservations with confirmation
  • View and manage existing reservations
